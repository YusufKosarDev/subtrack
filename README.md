# 💸 SubTrack

> Subscription and recurring expense tracker — never lose track of your monthly costs again.

SubTrack helps you visualize your subscriptions (Netflix, Spotify, gym memberships, cloud storage...) and alerts you before trials end or renewals hit.

## ✨ Features

- 📊 Visualize monthly & yearly costs with beautiful charts
- 🔔 Get notified about upcoming renewals and trial endings
- 🎨 Apple-style minimalist UI with glassmorphism & dark mode
- 💰 Multi-currency support with automatic conversion
- 📈 Track spending trends over time

## 🛠️ Tech Stack

**Frontend**
- React 18 + TypeScript + Vite
- TailwindCSS + shadcn/ui
- TanStack Query + Zustand
- Recharts

**Backend**
- Node.js + Express + TypeScript
- PostgreSQL + Prisma ORM
- Redis (caching, queues)
- JWT authentication

**DevOps**
- Docker + docker-compose
- GitHub Actions (CI/CD)
- Vitest + Playwright
