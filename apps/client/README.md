# SubTrack — Frontend

React + Vite + TypeScript frontend for SubTrack, a subscription and recurring expense tracker.

## Tech Stack

- React 19
- Vite 7
- TypeScript
- TailwindCSS v4
- shadcn/ui

## Scripts

- `npm run dev` — start dev server (port 5173)
- `npm run build` — production build
- `npm run lint` — run ESLint
- `npm run preview` — preview production build

See the root README for full project setup.
